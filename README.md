# Democracy-Bot
디스코드 사다리타기, 투표기능 봇

### 투표 봇
* 간단한 투표 기능을 가진 봇
* 모든 작업을 텍스트로 처리
* 명령어
  * !투표설정 (0 or 1) (후보1)/(후보2)/.../(후보n)
    * 투표를 설정함
    * 공개 투표시 !투표설정 0
    * 비공개 투표시 !투표설정 1
    * 이전의 투표가 끝나지 않았다면 강제로 종료함
  * !투표참여
    * 투표에 참여함
    * ID값을 넘겨받음
  * !투표 (INDEX)
    * 투표함
  * !투표종료
    * 투표를 종료한 후
